

### First clone the repo: 
```
git clone https://github.com/AashrayBenjamin/justrust.git
```
### To Run on linux  Open Terminal and run
   * Make sure you have rust installed
   * Then navigate to the path and run
   <br></br>  
   ```
   cargo run
   ```
<br></br>
